使用说明：

安装步骤：

将所有文件放在Web服务器目录下（如 htdocs 或 www）
修改 config.php 中的数据库连接信息
运行 database.sql 创建数据库和表
访问 index.php 开始使用
功能说明：

✅ 增加：填写标题和内容后点击"创建新剪贴板项"
✅ 删除：点击每项的"删除"按钮
✅ 修改：点击"编辑"按钮进入编辑模式
✅ 列表展示：自动显示所有未过期的项
✅ 自动过期：数据在创建5小时后自动过期，访问页面时会自动清理
可选：设置定时任务自动清理


# Linux crontab，每小时执行一次
`0 * * * * php /path/to/cron_cleanup.php`
